## Description
A "War" card game.

## Requirments
* Linux machine
* C++ libs
* Clang++14
* Clang-Tidy
* Valgrind

## Buliding
```
# Cloning the repo to local machine
git clone https://github.com/RoySenpai/sw_cpp_hw2b.git

# Bulding all the necessary files & the main programs
make demo

# Building the tester
make test1
```

## Running
```
# Runs a demo of the Card game program
./demo

# Runs a test of the program
./test1
```
